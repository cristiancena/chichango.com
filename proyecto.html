Proyecto

Presupuesto para el desarrollo de sitio web.

Necesidades

  • Comunicar las actividades, eventos y demás intereses de la gestión, siendo la web un eslabón más dentro de un ambiente de piezas de comunicación.
  • Publicar artículos con interacción:
    • hacia dentro: Posibilidad de comentar los artículos usando un sistema de comentarios o una red social.
    • fuera del sitio:
      • Compartirlos en redes sociales.
      • Enviar artículos por correo.

Conjunto de tecnologías (stack) que vamos a usar:

  • Backend LAMP: Linux, Apache, PHP y MySQL.
  • Frontend HTML5: HTML, CSS, Javascript.
  • CMS: Wordpress.
  • Sistema Operativo del Servidor: Ubuntu server 12.04.

Tiempo de desarrollo

30 días.

Para empezar se requiere:

  • Logotipo, o identidad corporativa, o manual de marca.
  • Material audio-visual o piezas de comunicación existentes.
  • Puede reunir algunos sitios web de ejemplo a los fines de comunicar sus intereses y preferencias. Realizar esta observación en fase de diseño evita rediseños o mejoras en fase de desarrollo, lo cual facilita el cumplimiento exacto del plazo de entrega del trabajo.
